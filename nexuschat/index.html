macOS 安裝教學

首次安裝需要額外一個步驟，請按以下指示操作

🚫

macOS 會顯示「App 已損壞」或「無法開啟」

由於 NexusChat 未通過 Apple 收費公證，macOS 會封鎖。請跟隨以下步驟解除限制，只需做一次，之後正常使用。

1

下載並安裝

下載 NexusChat.dmg，打開後將 NexusChat 圖示拖入 Applications 資料夾

2

🔑 解除封鎖（必做）

✅ 最可靠方法 — Terminal 指令

打開 Terminal（Spotlight 搜尋「Terminal」），貼上以下指令並按 Enter：

xattr -cr /Applications/NexusChat.app

指令完成後（沒有任何輸出是正常的），直接雙擊 NexusChat 開啟即可

方法二 — 右鍵開啟

在 Applications 找到 NexusChat → 右鍵點按（或 Control + 點按）→ 選「開啟」→ 彈出視窗再按「開啟」確認

方法三 — 系統設定

先嘗試雙擊開啟（即使失敗）→ 去「系統設定 → 私隱與安全性」→ 頁底找到 NexusChat 相關提示 → 按「仍要開啟」

✓

完成！之後正常使用

解除封鎖後，NexusChat 會正常啟動。之後每次直接雙擊開啟即可。App 會隨開機自動啟動，並在背景自動更新。

🖥 Windows 安裝說明
  1. 1. 下載 NexusChatSetup.exe
  2. 2. 執行安裝程式
  3. 3. 若出現 SmartScreen 警告，點按「更多資訊」→「仍要執行」
  4. 4. 安裝完成後自動啟動
