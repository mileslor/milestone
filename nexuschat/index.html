macOS 安裝教學

首次安裝需要額外一個步驟，請按以下指示操作

⚠️

重要提示：直接雙擊開啟會彈出「無法開啟」錯誤

由於 NexusChat 暫未通過 Apple 公證，macOS 會阻止直接執行。請跟隨以下第 3 步，用右鍵方式開啟，只需做一次。

1

下載 NexusChat.dmg

按上方「Download for macOS」按鈕，下載完成後在 Downloads 資料夾可以找到 NexusChat.dmg

2

打開 DMG，將 App 拖入 Applications

雙擊 NexusChat.dmg 開啟，將 NexusChat 圖示拖到 Applications 資料夾圖示上

📦 📁 Applications
3

🔑 關鍵步驟：用右鍵開啟（只需做一次）

去 Applications 資料夾，找到 NexusChat，切勿雙擊，請：

方法一

右鍵點按圖示

在 NexusChat 圖示上按右鍵（或 Control + 點按）→ 選擇選單最頂的「開啟」

方法二

系統設定解鎖

如彈出「無法驗證」視窗 → 去「系統設定 → 私隱與安全性」→ 頁面底部按「仍要開啟」

✓

完成！之後正常使用

確認開啟後，NexusChat 會正常啟動。之後每次直接雙擊開啟即可，不需要再做右鍵步驟。App 亦會隨開機自動啟動。

🖥 Windows 安裝說明
  1. 1. 下載 NexusChatSetup.exe
  2. 2. 執行安裝程式
  3. 3. 若出現 SmartScreen 警告，點按「更多資訊」→「仍要執行」
  4. 4. 安裝完成後自動啟動
