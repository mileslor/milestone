KaraQueue

本地 WiFi KTV 點歌系統

免費開源工具。手機掃 QR 即可點歌，電視大屏播放，AI 自動建立歌手歌庫，無需月費，自己架設。

GitHub 原始碼 / 下載 免費下載

📦 安裝說明

🍎 Mac：將 .dmg 安裝後把 App 拖入 Applications，然後開啟 Terminal 執行：
xattr -cr "/Applications/KaraQueue TV.app"再 double-click 開啟即可。

🪟 Windows：執行 .exe 安裝程式，若 SmartScreen 出現警告，按「更多資訊」→「仍要執行」即可。

📦 Installation Notes

🍎 Mac：Open the .dmg, drag the app to Applications, then open Terminal and run:
xattr -cr "/Applications/KaraQueue TV.app"Then double-click to open.

🪟 Windows：Run the .exe installer. If SmartScreen appears, click "More info" → "Run anyway".

功能特點

簡單架設，即刻開唱

📱

手機掃碼點歌

手機掃 QR Code 即可加入點歌，無需安裝任何 App，支援多人同時使用。

🤖

AI 自動建立歌庫

播放 2 分鐘後 AI 自動分析歌手、語言及年代，建立可瀏覽的本地歌庫（需 AI API Key）。

📺

大屏電視播放

電視畫面支援 Android TV APK，透過本地 WiFi 實現大屏播放，音質畫質由 YouTube 保證。

🔒

資料完全私有

Node.js 本地伺服器，所有歌單資料只存在您自己的設備，不上傳任何資訊。

🎵

完整播放控制

支援插播、調音量、上下首、重播，所有操作即時同步到電視。

💻

跨平台支援

支援 Mac / Windows / Linux，只需安裝 Node.js 即可運行，無需其他依賴。
