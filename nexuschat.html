為什麼選擇 NexusChat？

專為同時使用多個即時通訊帳號的用戶設計

👥

多帳號同時在線

工作、私人帳號同時在線，一鍵切換。各帳號完全獨立 Cookie 及 Session，互不干擾。

🔔

統一通知管理

所有帳號新訊息統一顯示系統通知，Dock / 工作列 badge 顯示未讀總數，一目了然。

🔄

自動斷線重連

網絡不穩時自動重連，無需手動刷新，確保持續在線。重連狀態即時顯示。

🌙

原生深色介面

全原生深色模式設計，護眼舒適。記憶視窗大小及位置，每次開啟回到上次狀態。

⚡

自動背景更新

背景自動偵測並下載最新版本，無需手動更新，確保功能及安全性持續跟進。

🔒

本地資料安全

所有登入 Session 儲存於本機，不上傳任何個人資料至第三方伺服器。

安裝說明

macOS

  1. 1. 下載 NexusChat.dmg
  2. 2. 打開 DMG，將 NexusChat 拖入 Applications
  3. 3. 首次開啟：右鍵點按 → 開啟（繞過 Gatekeeper）
  4. 4. 之後正常雙擊開啟即可
🖥

Windows

  1. 1. 下載 NexusChatSetup.exe
  2. 2. 執行安裝程式
  3. 3. 若出現 SmartScreen 警告，點按「仍要執行」
  4. 4. 安裝完成後自動啟動
⚠️ 注意： NexusChat 為獨立第三方應用程式，與 Meta、Telegram 或 Facebook 無關。使用本軟件前請自行評估相關平台服務條款。
